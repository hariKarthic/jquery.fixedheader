A simple utility plugin 
============================

jquery.fixedheader.js
============================

Purpose / Use :
---------------

To set the header of a large table to be fixed on screen when the table s scrolled vertically .


Requirements : 
---------------

>jquery v1.7


Usage :
---------------

call $(selector).fixedheader() on the tables that needs header to be fixed when scrolling.


options:
------------

fixedRows : 1 (default) - Sets the first row to be fixed.
